NABIL - Personal Portfolio (Static HTML/CSS/JS)
================================================

Struktur folder:
  index.html
  style.css
  script.js
  assets/    (semua gambar)

Cara menjalankan:
1. Extract seluruh isi ZIP ke satu folder.
2. Klik dua kali file "index.html" untuk membuka di browser.
   (atau klik kanan > Open with > Chrome/Edge/Firefox)

Cara mengganti gambar:
- Ganti file di folder /assets/ dengan nama yang sama:
    hero-1.jpg ... hero-4.jpg  -> slider hero
    portrait.jpg               -> foto About
    project-1.jpg ... project-3.jpg -> gambar proyek

Cara edit konten:
- Buka index.html di VS Code, ubah teks pada bagian Hero, About,
  Projects, Experience, dan Contact.
- Warna & style diatur di style.css (lihat variabel :root).
